## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->

O FinTech é um projeto de landing page/plataforma desenvolvido como parte prática do curso de Desenvolvedor Front-End.

O principal objetivo deste projeto foi aplicar, na prática, os conceitos de desenvolvimento web, design responsivo e estruturação de código. Ele nasceu da necessidade de consolidar o aprendizado em HTML e CSS, simulando um cenário real de entrega para uma agência digital.
